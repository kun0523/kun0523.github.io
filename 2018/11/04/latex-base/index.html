标记公式：

行内公式：

行内公式使用两个美元符包裹：

上下标：

  1. ^ 表示上标；
  2. _ 表示下标；
  3. {} 表示括号内的内容为一个整体
    ​ :$x^{y^z} = (1+e^x)^{-2xy^w}$
  4. 同时标上下标签：
    ​ - : $wi^j$
    ​ - $$w    {ij}^\prime$$ : $w_{ij}^\prime$

求和与求导：

  1. 求和：
    ​ - ： $f(u) = f(\sum{i=1}^Nx_i)$
    ​ - $$$f(u) = f(\sum    {i=1}^Nxi)f(u) = f(\sum{i=1}^Nx_i)$$
  2. 求导：
    ​ - : $\frac{\mathrm{d}f}{\mathrm{d}t}$
    ​ - : $\frac{\partial f(u)}{\partial u}$

分属表示：

  1. \frac{分子}{分母}
    : $\frac{a+b}{d+c}$
  2. 分子 \over 分母
    : ${a+b} \over {d+c}$

各种括号的使用：

符号名称 LaTex代码 样例 效果
小括号 () \left( x \right) $\left( x \right) $
中括号 [] \left[ \frac12 \right] $\left[ \frac12 \right] $
大括号 {} { \frac12 } ${ \frac12 } $
尖括号 \langle \rangle \langle \frac12 \rangle $\langle \frac12 \rangle$
向上取整 \lceil \rceil \lceil \frac12 \rceil $\lceil \frac12 \rceil $
向下取整 \lfloor \rfloor \lfloor \frac12 \rfloor $\lfloor \frac12 \rfloor $
  • 注意：\left \right 必须成对出现，当需要只使用一侧的情况，使用 . 代表为空的情况：
    \left . \frac{du}{dx} \right| _{x=0} : $\left . \frac{du}{dx} \right|_{x=0}$

根号表示：

  1. \sqrt[3]{\frac xy} : $\sqrt[3]{\frac xy}$
  2. \sqrt{x^3} : $\sqrt{x^3}$

省略号的使用：

  1. 对齐底线的省略号： \ldots;
  2. 对齐中间位置的省略号： \cdots;
    f(x_1, x_2, \ldots, x_n) = x_1^2 + x_2^2 + \cdots + x_n^2 : $f(x_1, x_2, \ldots, x_n) = x_1^2 + x_2^2 + \cdots + x_n^2$
